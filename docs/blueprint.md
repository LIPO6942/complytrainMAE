# **App Name**: ComplyTrain

## Core Features:

- Content Management: Supports SCORM 1.2, SCORM 2004 and xAPI content. Includes a quiz creator for QCM, true/false, and drag-and-drop questions.
- User Management: Secure registration and role-based access control (admin, trainers, learners, BU, auditors).
- Adaptive Quizzes and Exams: Offers timed quizzes, final exams, and anti-cheating measures such as question permutation and tab restrictions.
- Compliance Reporting: Real-time analytics dashboard for success rates, completion rates, and risk matrices, with automated export to Excel and PDF.
- AI Compliance Tutor: An AI tool provides learners with a tutor, automated summaries, and personalized risk scoring.
- Gamified Learning Paths: Uses badges and compliance challenges to motivate the learners
- Secure Audit Trails: Provides audit trails using hashed, timestamped certificates.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and professionalism.
- Background color: Very light gray (#F5F5F5), a desaturated tint of the primary color.
- Accent color: Teal (#008080), to give contrast.
- Body and headline font: 'Inter' sans-serif, for a modern and neutral look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean and professional icons for navigation and compliance indicators.
- Subtle micro-animations for progress tracking, validation, and alerts.
